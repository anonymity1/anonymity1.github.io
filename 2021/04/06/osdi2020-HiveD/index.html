HiveD

HiveD: Sharing a GPU Cluster for Deep Learning with Guarantees (OSDIвҖҷ 2020)пјҢеҢ—еӨ§пјҢMicrosoftпјҢжёҜеӨ§еҗҲдҪңзҡ„дёҖзҜҮж–Үз« гҖӮ

еӨҡз§ҹжҲ·еңЁGPUйӣҶзҫӨдёҠиҝӣиЎҢDLи®ӯз»ғжҳҜдёҖз§Қиҫғдёәеёёи§Ғзҡ„еңәжҷҜпјҢз„¶иҖҢзӣ®еүҚGPUйӣҶзҫӨиө„жәҗеҲҶй…ҚжҳҜеҹәдәҺquotaзҡ„пјҢеҚіжҜҸдёӘд»»еҠЎжҢҮе®ҡйңҖиҰҒзҡ„GPUж•°зӣ®пјҢиҝҷж ·зҡ„зІ—зІ’еәҰиө„жәҗи°ғеәҰж–№ејҸе®№жҳ“йҖ жҲҗжӣҙй•ҝзҡ„зҙҜз§Ҝе»¶иҝҹпјҢжң¬ж–Үеӣ жӯӨжҸҗеҮәдәҶдёҖз§Қжӣҙз»ҶзІ’еәҰGPUйӣҶзҫӨиө„жәҗи°ғеәҰж–№ејҸгҖӮ

еҠЁжңә

иҝҷжҳҜеӣ дёәзҺ°жңүеңЁе…¬е…ұGPUйӣҶзҫӨдёҠдёәеӨҡз§ҹжҲ·еҲҶй…Қи®ӯз»ғиө„жәҗзҡ„ж–№ејҸжҳҜеҹәдәҺquotaзҡ„пјҢеҚіеҲҶй…ҚGPUзҡ„ж•°зӣ®гҖӮе…¬е…ұGPUйӣҶзҫӨи°ғеәҰеҷЁдёәз§ҹжҲ·е’Ңз§ҒжңүйӣҶзҫӨйҮҢдёҖж ·еӨҡзҡ„GPUж•°зӣ®пјҢдҪҶжҳҜиҝҷж ·зҡ„GPUж•ҲзҺҮеҸҜиғҪз”ұдәҺжҹҗдәӣеҺҹеӣ пјҲеҚ•дёҖnodeи®Ўз®—иғҪеҠӣдёҚејәпјҢжІЎжңүи¶іеӨҹGPUпјүйҖ жҲҗеҪ“еүҚз§ҹжҲ·д»»еҠЎе®ҢжҲҗж—¶й—ҙеўһеҠ пјҢд»ҺиҖҢдә§з”ҹйҳҹеҲ—зҙҜз§Ҝе»¶иҝҹгҖӮ

е…¬е…ұGPUйӣҶзҫӨдёҠеҸҜиғҪдјҡжңүеӨҡдёӘnodeпјҢжҜҸдёӘnodeдёҠжңүеӨҡдёӘGPUгҖӮеҰӮжһңдёҖдёӘDLд»»еҠЎйңҖиҰҒ64дёӘGPUпјҢйӮЈд№ҲйӣҶзҫӨи°ғеәҰеҷЁдә§з”ҹдёҖдёӘ8nodeпјҢжҜҸдёӘnodeеҲҶй…Қ8GPUзҡ„еҜ№еә”affinityгҖӮеҗҢж—¶дјҡжңүдәӣеҫҲе°Ҹзҡ„DLд»»еҠЎпјҢиҝҷдәӣд»»еҠЎеҸҜиғҪеҸӘйңҖиҰҒ1-3дёӘGPUпјҢдҪҶжҳҜиҝҷж ·зҡ„DLд»»еҠЎд№ҹдјҡеҚ жҚ®дёҖдёӘnodeпјҢз”ұдәҺеҫҲеӨҡзҡ„nodeиў«иҝҷж ·зҡ„е°ҸDLд»»еҠЎеҚ жҚ®пјҢйӣҶзҫӨи°ғеәҰеҷЁж— жі•дёәйңҖиҰҒ64дёӘGPUзҡ„DLд»»еҠЎзҡ„affinityжүҫеҲ°еҜ№еә”зҡ„й…ҚзҪ®пјҢжүҖд»ҘиҝҷдёӘеӨ§DLд»»еҠЎеҸӘиғҪзӯүеҫ…пјҢз”ұжӯӨдә§з”ҹдәҶзҙҜз§Ҝе»¶иҝҹгҖӮ

иҝҷдёӘй—®йўҳдјјд№ҺжҳҜеҸҜд»Ҙиў«и§ЈеҶізҡ„пјҢжңүдәӣзұ»дјјдәҺеҶ…еӯҳз®ЎзҗҶдёӯзҡ„memory fragmentationй—®йўҳгҖӮ

пјҲиҝҷдёӘеҠЁжңәдјјд№ҺжңүзӮ№easyпјүпјҢдёҚиҝҮеә•еұӮиө„жәҗжІЎжңүжұ еҢ–пјҢиҖҢжҳҜеҲҶеұӮеӯҳеңЁпјҢзЎ®е®һжҳҜдёҖдёӘзҺ°е®һзҡ„й—®йўҳгҖӮ

и§ЈеҶіж–№жЎҲ

жң¬ж–ҮжҸҗеҮәдәҶHiveDзі»з»ҹпјҢзЎ®дҝқе…ұдә«GPUжү§иЎҢж—¶жҳҜsafetyзҡ„гҖӮHiveDе°ҶGPUиө„жәҗеҲҶдёәVirtual Private ClusterпјҲVCпјүе’Ңphysical ClusterдёӨеұӮгҖӮHiveDе°ҶжҜҸдёӘз§ҹжҲ·иЎЁзӨәдёәдёҖдёӘVCпјҢжҜҸдёӘVCдјҡйў„е…Ҳе®үжҺ’дёҖзі»еҲ—зҡ„cellпјҢиҝҷйҮҢзҡ„cellиЎЁзӨәйңҖиҰҒеӨҡе°‘иө„жәҗгҖӮ

cellйҮҢзҡ„иө„жәҗжҳҜеҲҶеұӮзҡ„пјҢиҝҷзҜҮж–Үз« дҪңиҖ…е°Ҷиө„жәҗеҲҶдёә5еұӮпјҡGPU -> PCIe switch -> CPU socket -> node -> RackпјҢйҖҡиҝҮеҲҶеұӮе®һзҺ°з»ҶзІ’еәҰиө„жәҗи°ғеәҰгҖӮпјҲиҝҷдёӘеҲҶеұӮжңүдәӣз–‘жғ‘пјҢеҗҢж—¶жҜҸдёҖеұӮеҶ…еҗ„дёӘcellд№Ӣй—ҙзҡ„йҖҡдҝЎжҒҗжҖ•д№ҹйңҖиҰҒиҖғиҷ‘пјү

еҰӮжһңе°Ҷиө„жәҗи®ӨдёәжҳҜжұ еҢ–зҡ„пјҢдёҖдәӣеёёи§„зҡ„и°ғеәҰеҷЁпјҢжҜ”еҰӮk8s default schedularдјҡж №жҚ®д»»еҠЎйңҖиҰҒиө„жәҗзҡ„ж•°йҮҸзӣҙжҺҘеҲҶй…Қиө„жәҗпјҢдҪҶжҳҜеӣ дёәGPUйӣҶзҫӨйҮҢиө„жәҗжІЎжңүжұ еҢ–пјҢжүҖд»ҘйңҖиҰҒеҶҚе°ҒиЈ…дёҖеұӮпјҢе°Ҷnodeд»ҘеҸҠжӣҙз»ҶзІ’еәҰзҡ„иө„жәҗжұ еҢ–пјҢиҝҷд№ҹжҳҜиҝҷзҜҮж–Үз« зҡ„ж ёеҝғideaгҖӮ

еҗҢж—¶з”ұдәҺеҲҶеұӮиө„жәҗзҡ„еЎ”ејҸз»“жһ„пјҢеә•еұӮиө„жәҗдёҺдёҠеұӮиө„жәҗжңүйҡ¶еұһе…ізі»пјҢйңҖиҰҒж №жҚ®иҝҷз§Қйҡ¶еұһе…ізі»и°ғж•ҙеҲҶй…Қзҡ„з»ҶзІ’еәҰиө„жәҗгҖӮиҝҷзҜҮж–Үз« з§°д№Ӣдёәbuddy cellпјҢеҪ“жүҖжңүзҡ„buddy cellйғҪжҳҜз©әй—Ізҡ„ж—¶еҖҷпјҢйӮЈд№Ҳе°ұеҸҜд»ҘеҗҲе№¶жҲҗдёҖдёӘжӣҙзІ—зІ’еәҰзҡ„иө„жәҗпјҢеңЁеҲҶй…Қз»ҶзІ’еәҰзҡ„иө„жәҗж—¶пјҢйҖҡиҝҮиҖғиҷ‘иҝҷз§Қйҡ¶еұһе…ізі»пјҢе°ҪеҸҜиғҪдҝқжҢҒзІ—зІ’еәҰзҡ„иө„жәҗжІЎжңүиў«еҚ з”ЁпјҢд»ҺиҖҢдҝқиҜҒеӨ§зҡ„DLд»»еҠЎеҸҜд»Ҙиў«жӯЈеёёжү§иЎҢиҖҢдёҚз”Ёе»¶иҝҹгҖӮ

жңҖеҗҺжҳҜе…ідәҺдҪҺдјҳе…Ҳзә§VCпјҢиҝҷзұ»VCеҸҜд»Ҙиў«йҡҸж—¶жҠўеҚ пјҢжүҖд»ҘеҲҶй…Қcellж—¶пјҢдјҳе…ҲеҲҶй…ҚеҲ°йӮЈдәӣжІЎжңүиў«еӨ§DLд»»еҠЎеҚ жҚ®зҡ„buddy cellеҪ“дёӯпјҢиҝҷж ·еҸҜд»ҘдҝқиҜҒеӨ§жҰӮзҺҮдёҚиў«жҠўеҚ пјҢеҗҢж ·еҜ№дәҺguaranteed VCпјҢдјҳе…ҲеҲҶй…Қbuddy cellйҮҢжІЎжңүдҪҺдјҳе…Ҳзә§д»»еҠЎзҡ„VCпјҢд»ҺиҖҢйҷҚдҪҺжҠўеҚ еҸ‘з”ҹзҡ„ж¬Ўж•°гҖӮ

е®һйӘҢзҺҜеўғ

2019е№ҙ11жңҲе®һйҷ…йғЁзҪІеңЁopenPAIйЎ№зӣ®дёӯпјҢйӣҶзҫӨеҪ“дёӯжңү800еӨҡеҸ°GPUпјҢNvidiaе’ҢAMDзҡ„пјҢд№ҹеҢ…жӢ¬200еӨҡеҸ°Azure GPUиҷҡжӢҹжңәгҖӮжү§иЎҢзҡ„д»»еҠЎеҢ…жӢ¬NLPд»»еҠЎпјҲBERTпјүпјҢAutoMLе®һйӘҢпјҢд»ҘеҸҠеҮ зҷҫдёӘеҚ•GPUд»»еҠЎгҖӮ

йЎ№зӣ®жһ„жҲҗпјҡ7700иЎҢgolangд»Јз ҒпјҢд»ҘеҸҠjsпјҢshellи„ҡжң¬е’Ңyamlй…ҚзҪ®ж–Үд»¶гҖӮ

е·ҘдҪңжөҒпјҡе’Ңk8sзҡ„й»ҳи®Өи°ғеәҰеҷЁй…ҚеҗҲе·ҘдҪңпјҢиғҪеӨҹеӨҚз”Ёk8sи°ғеәҰеҷЁзҡ„еҹәжң¬йҖ»иҫ‘е’Ңзӣёе…ій…ҚзҪ®гҖӮ
